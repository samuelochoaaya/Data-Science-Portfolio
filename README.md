## 🚀 Proyectos

### 1. 🏦 Predicción de Churn Bancario

**¿De qué trata?**
Modelo de machine learning para predecir qué clientes abandonarán un banco, 
permitiendo acciones proactivas de retención.

**Técnicas usadas:**
- Machine Learning (Random Forest, ADABOOST, XGBOOST)
- Validación cruzada
- Optimización de hiperparámetros
- Análisis de métricas (Recall, Precision, F1)

**Resultados:**
- Mejor modelo: XGBOOST
- Recall: 0.34 (detecta 34% de churners)
- F1-Score: 0.31
